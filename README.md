## About The Project

This project was created with the project [![LAB Cloathing Collection](https://github.com/MatheusFilipeFreitas/LAB-Clothing-Collection)]. The LAB Fashion API is a solution that helps all the users to easily manage the cration of collections and models for their collections. This project contains High level of security, with cryptographing the users password, and for using this API each user need to be looged.
Project was introduced by [![DEVinHouse - Audaces](https://github.com/DEVin-Audaces-V2)].

## Getting Started

For installing this project you just need to download the project to your PC, and configure the Database string in the appsettings.json. Also be sure to already had installed the .NET SDK 7 and SQL Server.

### Prerequisites

* dotnet --version
  ```sh
  7.0.304
  ```
* sql --version
  ```sh
  2022 
  ```
